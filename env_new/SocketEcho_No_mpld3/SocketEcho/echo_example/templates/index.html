我們專致於三個領域的程式開發:
1. 利用 API 下載每一筆交易所的 Ticktime 資料, 包含外期 (芝加哥交易所等) 及台期 (大小台, 股票) 的資料下載與程式自動下單的客製化. 並利用 Websocket 將即時資訊呈現於網頁中.
2. Android 平台的 App 開發. 目前已與數個關於 Bluetooth Low Energy 及 NFC 智慧鎖的相關應用.
3. Windows / Linux 平台的程式客製化. 包含資料庫, 網路蜘蛛人, 圖形辨識及 Offic 自動化等應用.
